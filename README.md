# Tanakorn Sawatna Resume

Static resume website for Tanakorn Sawatna.

## Files

- `index.html` - resume content and page structure
- `style.css` - layout, colors, responsive styles, and icon styling
- `images/photo-resume.JPG` - profile photo used by the resume page

## Run locally

Open `index.html` directly in a browser.

## Edit content

Update the resume text in `index.html`:

- name and profile summary
- contact details
- skills and tools
- education
- experience and projects

Update visual styling in `style.css`.

## Publish with GitHub Pages

1. Open repository settings on GitHub.
2. Go to Pages.
3. Set source to the `main` branch.
4. Save and wait for GitHub Pages to publish the site.

## Notes

This project has no build step and no package dependencies.
