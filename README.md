# Tanakorn Sawatna Resume

Static resume website built with HTML and CSS.

## Open the website

Open `index.html` directly in a browser.

## Project structure

- `index.html` - resume content and page structure
- `style.css` - responsive glassmorphism design
- `images/photo-resume.JPG` - profile image

## Git workflow

Use a separate branch for every change.

Recommended branch names:

- `codex/redesign-glass-resume`
- `codex/update-content`
- `codex/fix-responsive-layout`

Use clear messages for commits and merges.

Examples:

```bash
git commit -m "Redesign resume website with glass layout"
git merge codex/redesign-glass-resume -m "Merge glass resume redesign"
```
